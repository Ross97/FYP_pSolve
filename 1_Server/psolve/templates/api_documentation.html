pSolve API Documentation


API information

http://ross-dev.live:5000/ is the base URL for all queries


Sudoku Detection

/sudoku/detect
Usage: Send base-64 encoded images of sudoku puzzles to obtain the detected digits
Parameters: b64_data (required)
Returns: result (always), digits, filename

Python 3 example using the requests library

import requests
url = 'http://ross-dev.live:5000/sudoku/detect'
my_sudoku_img = {'b64_data': MY_B64_ENCODED_IMAGE}
response = requests.post(url, json=my_sudoku_img)
print(response.status_code, response.text)


Sudoku Solver

/sudoku/solve
Usage: Send a 9*9 array of digits in an unsolved sudoku to obtain the solution
Parameters: sudoku (required), filename (optional)
Returns: result (always), solution

Python 3 example using the requests library

import requests
url = 'http://ross-dev.live:5000/sudoku/solve'
my_sudoku = {'sudoku': [[5,3,0,0,7,0,0,0,0],[6,0,0,1,9,5,0,0,0],[0,9,8,0,0,0,0,6,0],[8,0,0,0,6,0,0,0,3],[4,0,0,8,0,3,0,0,1],[7,0,0,0,2,0,0,0,6],[0,6,0,0,0,0,2,8,0],[0,0,0,4,1,9,0,0,5],[0,0,0,0,8,0,0,0,9]]}
response = requests.post(url, json=my_sudoku)
print(response.status_code, response.text)


Spot The Difference

/differences/detect
Usage: Send two same-sized base-64 encoded images to obtain a filename showing the differences.
Parameters: image_one (required), image_two (required), is_camera_image (optional)
Returns: result (always), filename

Python 3 example using the requests library

import requests
url = 'http://ross-dev.live:5000/differences/detect'
my_images = {'image_one': MY_B64_ENCODED_IMAGE_1, 'image_two': MY_B64_ENCODED_IMAGE_2}
response = requests.post(url, json=my_images)
print(response.status_code, response.text)
